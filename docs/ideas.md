# 1. Course Structure & Pedagogy

## Lesson Anatomy (working model)
- Core vocabulary list
- Main reading using that vocabulary
- Glosses for additional words not yet in vocab
- Grammar lesson (focused, incremental)
- Practice exercises
- Cultural / historical “interest feature”
- Additional authentic Greek passage (e.g., Xenophon)

## Introductory Unit (separate from normal lessons)
- Greek alphabet
- Pronunciation (audio-heavy)
- Diphthongs
- Breathings
- Accents and diacritics
- Early decoding exercises

Goal:
→ Move students quickly into real reading without killing enthusiasm

---

# 2. Learning Experience Design

## Core Learning Loop (to be refined)
1. Learn vocabulary
2. Hear vocabulary (audio)
3. Read passage
4. Use glosses
5. Study grammar
6. Practice (varied formats)
7. Get feedback
8. Repeat weak areas

## Key principle
- Repetition + variation = retention
- Immediate feedback is essential

---

# 3. Practice & Assessment Ideas

## Exercise Types
- Multiple choice (recognition)
- Matching (Greek ↔ English)
- Fill-in-the-blank
- Typing answers (recall)
- Parsing identification
- Reading comprehension

## Error Tracking
- If student gets something wrong:
  → show again soon
  → repeat until correct

Possible future:
- simple spaced repetition system
- track “new / learning / mastered”

# 4. Vocabulary Strategy

## Vocabulary Model
- Core vocabulary (must learn)
- Thematic vocabulary (lesson-specific)
- Glosses (temporary support, not required yet)

## Vocabulary Lifecycle (to be defined)
- Introduced → Practiced → Reinforced → Mastered

## Flashcards

### Hybrid approach
- Website:
  - authoritative vocabulary
  - correct pronunciation audio
- Quizlet:
  - optional external practice
  - mobile-friendly repetition

Important:
→ Quizlet audio is NOT reliable for Ancient Greek

# 5. Audio & Pronunciation

## Core Idea
Audio is a primary feature, not optional

## Features
- Word-level pronunciation
- Sentence audio
- Full passage audio
- Optional slow playback

## Key Principle
- Website audio = authoritative pronunciation
- Must be consistent across entire system

# 6. Video / Multimedia

## Possible Uses
- Introductory unit (alphabet, pronunciation)
- Cultural/history segments
- “Interest feature” sections

## Avoid
- Overuse in grammar instruction
- Passive-only learning

Guideline:
→ Video = engagement, not core learning

# 8. Engagement & Motivation

## Challenges
- Early lessons (alphabet, accents) can reduce excitement

## Possible Solutions
- Treat intro material as “Unit 0”
- Keep it short and interactive
- Introduce real Greek reading as early as possible
- Use audio to create immersion
